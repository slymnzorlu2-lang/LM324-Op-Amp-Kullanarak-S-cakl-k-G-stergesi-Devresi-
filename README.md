# LM324 Op-Amp Tabanlı Sıcaklık Göstergesi Devresi

Bu proje, Altium Designer kullanılarak geliştirilmiş LM324 Op-Amp tabanlı bir sıcaklık göstergesi devresinin donanım dosyalarını içermektedir. 

## Proje İçeriği
* Altium kaynak dosyaları (.PrjPcb, .SchDoc, .PcbDoc)
* Şematik PDF dökümü
* 3 boyutlu (3D) PCB tasarımı görselleri
